# React Product Dashboard

## Description
This project is a simple React application created with Vite.  
It demonstrates basic React concepts such as components, props, state, controlled inputs, conditional rendering, and composition.

## File Structure
- App.jsx: Root component that manages application state
- ProductList.jsx: Renders a list of products using map
- Product.jsx: Displays a single product using props
- Card.jsx: Reusable layout component using children

## Challenges Faced
Understanding how to control UI visibility using state and how to pass data between components using props.
